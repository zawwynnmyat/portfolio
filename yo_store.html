Yo! Store

Yo! Store is a mobile application made for a client who is reselling phone bills and game items, such as diamonds, weekly-passes in Mobile Legends Bang Bang, UC in PUBG Moble, gems and gold-passes in Clash of Clans, diamonds in Freefire and genesis crystals in Genshin Impact, etc.

Core Features

  • The app allows users to sign-in or sign-up with Google.
  • Game ID validator for Mobile Legends Bang Bang
  • Easy to order game items
  • Order Tracking (According to my client, it must have 2 order statuses: “Pending” for pending orders and “Done” for orders which have already completed.
  • Order phone bills and internet data of 4 biggest operators in Myanmar: Ooredoo, ATOM, MPT and MyTel.
  • Search Games
  • Flash Sales
  • In terms of admin side, Admin can check and update order status, Search Game ID, Copiable Game IDs and easy to proceed, Add / Update / Delete item information, prices and discounts, Add / Update / Delete Payment Accounts

Tech Stack

  • Flutter for cross-platform development
  • Firebase as a backend service
