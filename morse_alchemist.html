Morse Alchemist

Morse Alchemist is a powerful app designed to translate Morse Code into text and vice versa. Whether you're learning Morse Code or need a reliable tool for quick translations, Morse Alchemist has you covered. It is the perfect companion for Morse Code enthusiasts, amateur radio operators, and anyone curious about this fascinating communication system.

Core Features

  • Translate Morse Code to Text and Text to Morse Code
  • Play Morse Code sounds for better learning and recognition
  • Explore Morse Code representations for English letters, numbers, and symbols
  • Access a comprehensive list of Q-Codes, abbreviations, and pro-signs

Tech Stack

  • SwiftUI
  • AVFoundation
