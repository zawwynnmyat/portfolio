Padonmar Book Store

Padonmar is an app developed for Padonmar book store based in Yangon, Myanmar.

Core Features

  • Explore books by category
  • Book Preview & Read English Books
  • Order Books
  • Search books by author, book title

Tech Stack

  • Flutter for cross-platform development
  • Firebase as a backend service
  • Redux for State Management
  • Hive for Local Database
