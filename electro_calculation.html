Electro Calculation

Electro Calculation is a mobile app designed to calculate some rudimentary circuits and component values. This is my first mobile app developed with flutter.

Using this app, you can calculate the following.

  • 3 OR 4 Digits SMD Resistor Codes
  • SMD Resistor EIA-96
  • Resistor Color Code (3 Bands)
  • Resistor Color Code (4 Bands)
  • Resistor Color Code (5 Bands)
  • Resistor Color Code (6 Bands)
  • Ceramic Capacitor Code
  • Battery Life
  • 555 Astable
  • 555 Monostable
  • LED Series Resistor Calculation
  • Op-amp (Non-Inverting & Inverting) Gain
  • Voltage Divider
  • Resistor in Series
  • Resistor in Parallel
