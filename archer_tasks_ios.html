Archer Tasks

“Archer Tasks” is a task manger or to-do-list application. It can be used to managed your tasks that have to be completed in folders, so called lists.

Core Features

  • Manage Your Tasks in Lists
  • Sort Your Lists and Tasks in your favorite order
  • Add or Edit tasks depending on your priorities, category and due date.
  • Supports Multiple Language
  • Customize App Icon

Tech Stack

  • SwiftUI
  • CoreData (Entity Relationship, Delete Rules, Filtering, Sorting, Fetching, Inserting, Editing, Deleting, Batch Deletion)
  • UserDefaults and AppStorage to store app settings
  • SceneStorage to store draft form’s textfield data
  • iOS Haptic Feedbacks
  • Dynamic App Icon (5 different app icons)
  • Dynamic App Localization (includes 11 Languages)
  • Blurring Items While Multitasking using App Lifecycle with ScenePhase (to obscure users’ todo tasks)
  • Launch Screen Storyboard
  • Onboarding Flow
  • TipKit to show tips in app fresh install
  • MFMailComposeView to get user feedback
